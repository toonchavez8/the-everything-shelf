# Lista para hospital (Cesárea – [**Santa Maria Chapalita** ](https://maps.app.goo.gl/5TrDDC8nVhZaBgj16))

## Para mamá

### Documentos
- Identificación oficial
- Estudios médicos y expediente
- Orden de ingreso / papeles del hospital
- Tarjeta del seguro / póliza
- Celular
- Cargador (preferente cable largo)
- Dinero en efectivo / tarjeta

### Ropa
- Pijama con botones (lactancia)
- Bata o robe
- Pantuflas o sandalias
- Calcetines
- Ropa cómoda para salida

### Higiene personal
- Cepillo de dientes
- Pasta dental
- Shampoo y acondicionador
- Jabón corporal
- Desodorante
- Cepillo o liga para cabello
- Toallas sanitarias nocturnas

### Recuperación cesárea
- Pañales para adulto
- Faja postparto (opcional)
- Almohada pequeña

### Lactancia
- Brasier de lactancia
- Protectores de lactancia
- Pezonera (opcional)
- Extractor de leche (opcional)

### Ropa
- 2 a 3 cambios de ropa (RN y 0–3 meses)
- Pijamas

- Mitones
- Calcetines

### Higiene
- Pañales para recién nacido y Etapa 1
- Toallas húmedas
- *Crema para rozaduras*

### Alimentación
- Biberones esterilizados (en bolsa ziploc)
- *Fórmula (solo si el pediatra lo indicó)*

### Extras
- Cobijita 
- Cobijita para mama
- manta

## Recomendaciones generales
- Separar en dos bolsas: parto y estancia
- Llevar pocos cambios de ropa para el bebé
- Llevar snacks ligeros
- Considerar extensión eléctrica o multicontacto
- Llevar carpeta organizada con documentos
